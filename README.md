# PlayerInfo
This plugin gives you the option to set up your own PlayerInfo

# Config
```yml
player_info:
  is_enabled: true
  # Tags: {nick} | {unit}
  player_info:
    ClassD: >-
      <color=#EE7600>{nick}

      Dboiii</color>
    Scp049: >-
      <color=#228B22>{nick}

      Doctor</color>
    Scp173: >-
      <color=#960018>{nick}

      Matthew</color>
    NtfCaptain: >-
      <color=#00FFFF>{nick}

      Captain {unit}</color>
```
